Alerts

Make it clear that an error has occurred (in language the user can understand), and that the user needs to do something. Buttons and links inside alerts should provide the user with a way to solve the problem.

Inform users of errors in a friendly way. Don’t make them feel bad for making a mistake. Also avoid exclamation marks – this can make the error look more significant than it really is.

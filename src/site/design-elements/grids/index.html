Grids

We use Bootstrap's 12-column responsive grid, on both desktop and mobile, and prebuilt grid classes for layouts that scale and stack based on device and viewport size. Columns allow us to adjust the placement and size of modules within layouts, making the best use of the space available at each breakpoint and maintaining hierarchy within layouts. To use the grid, wrap your content in a <div> within the .container class.

(Protip: Pages with text only should not be more than 10 columns wide or text lines become too wide and hard to read.)

We use the following Bootstrap media queriesto mark where there are big changes to our grid. This allows us to optimise the page layout for all screen widths across many devices.

We are currently using the following breakpoints:

  • Extra Small - 480px (Phones)
  • Small - 768px (Tablets)
  • Medium - 992px (Desktops)
  • Large - 1200px (Wide Screens)

Grids

We use Bootstrap's 12-column responsive grid and prebuilt grid classes for layouts that scale and stack based on device and viewport size. To use the grid with set max-width (960px), wrap your content in a <div> with the .container class.

This is two column layout.
It becomes one column layout on mobile.

You can find more grid examples at Bootstrap grid examples Bootstrap media queries are used to make content work in small, medium and large sizes.

(Protip: Pages with text only should not be more than 10 columns wide or text lines become too wide and hard to read.)

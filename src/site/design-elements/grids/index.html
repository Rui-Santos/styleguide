Grids

Bootstrap media queries are used to make content work in small, medium and large sizes.

We use Bootstrap's 12-column responsive grid and prebuilt grid classes for layouts that scale and stack based on device and viewport size. To use the grid, wrap your content in a <div> with the .container class.

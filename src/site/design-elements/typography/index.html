H1 should only be used as page title. A page should not have more than one H1.

H2 is used as page titles for sub-sections and post titles blog posts.

H3 and H4 are used regularly in blog posts as subheaders.

H5 is used on project, data and documentation pages and sometimes in blog posts as subheaders.

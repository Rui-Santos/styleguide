Colors

There are three primary brand colors that are derived from the logo.

Mapzen Red is use in page titles, large headers and primary call-to-action buttons. Outside of that, the color red carries a lot of meaning and should be used sparingly.

Mapzen Blue is the text link color and is sometimes use for buttons.

Mapzen Yellow should only be used as background color or for text over a dark background.

(Note: The hex values are for use outside of code like for design, presentations etc.)

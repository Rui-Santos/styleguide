Tooltip

Tooltip is a small pop-up box that appears when an element is hovered or focused. Tooltips with zero-length titles are never displayed.

Tooltips have the same problem that Popover component has. (Tooltip distorts the original element's margin.) To prevent this problem, do not use .btn-group wrapper for the series of buttons.

If you need to use a series of buttons with Tooltips, please use the buttons without .btn-group wrapper like below.

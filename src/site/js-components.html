JS Components

Some components will require JavaScript in order to function correctly. In addition, we assume that all of our pages will be measured by Google Analytics. So, we've bundled all of the baseline JavaScript requirements into one package: 

<script src="https://mapzen.com/common/styleguide/scripts/mapzen-styleguide.min.js"></script>

This file is about 35kb minified and gzipped. For best results, insert this script tag at the bottom of a webpage, just before the closing </body> tag, as opposed to in the <head>. You can see its preferred placement in the HTML Boilerplate.

These third-party libraries are included:

  • Bootstrap JavaScript (version 3.3.6) for these plugins: Transitions, Dropdown, Scrollspy, Collapse, and Affix. Other Bootstrap plugins are not supported by our styleguide at this time.
  • Google Analytics using Mapzen's Universal Analytics ID. We also include a helper function for tracking specific events (documentation forthcoming).

In addition, client-side JavaScript is used to make sure our top navigation bar works correctly, and to handle very wide tables. Specific design components that depend on these JavaScript files are described in greater detail below. If you need other JavaScript libraries that are not included in this list above, please include them in your page separately. (For instance, jQuery is not provided by the Styleguide.)
